OpenAuth Module
===============

Introduction
------------
Social Connect Module

Installation
------------

Using Git 
---------
You can install using native git submodules:

    git clone git@github.com:ledsinclouds/OpenAuth.git
    
Using Composer 
--------------

Add to composer And Install:

	"ledsinclouds/open-auth": "dev-master"  
	composer update
